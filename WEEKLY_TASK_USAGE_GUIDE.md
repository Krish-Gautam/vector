# Weekly Task Tracking - User Guide

## What's New?

Your dashboard now automatically maintains a **7-day rolling task plan**. You'll always see what's coming up for the entire week ahead!

## Features

### 1. **Automatic Weekly Plan**
- System automatically generates tasks for the next 7 days
- When you complete tasks, new ones are added to maintain the 7-day window
- No manual intervention needed - it just works!

### 2. **Visual Weekly Tracker**

The weekly tracker shows 7 days starting from today:

#### Status Indicators:
- **✓ Green Box** - All tasks completed for that day
- **Blue Box with Count** - Tasks scheduled (shows completed/total)
- **Gray Box** - No tasks scheduled for that day

#### Information Displayed:
- Day of week (Mon, Tue, Wed, etc.)
- Date number
- Task completion status
- Task count (e.g., "1/2" means 1 of 2 tasks done)

### 3. **Today's Plan Section**
- Shows all tasks scheduled for today
- Click any task to mark it complete
- Real-time updates as you complete tasks
- Shows estimated time for each task

### 4. **Manual Generation**
If you see "No weekly plan available":
1. Click the **"Generate Weekly Plan"** button
2. System will create tasks for the next 7 days
3. Dashboard automatically refreshes

## Understanding the System

### Task Generation Logic

1. **Initial Load**
   - System checks if you have tasks for next 7 days
   - Generates missing tasks automatically
   - Based on your roadmap and daily study time

2. **After Completion**
   - When you complete a task that finishes a roadmap item
   - System automatically generates new tasks
   - Maintains 7-day rolling window

3. **Task Scheduling**
   - Tasks are distributed across days
   - Based on your daily study minutes setting
   - Follows your roadmap order

### What Tasks Are Generated?

- **Source**: Tasks come from your active roadmap
- **Duration**: Based on your daily study time preference
- **Order**: Follows roadmap task sequence
- **Completion**: When a roadmap task is done, moves to next one

## Troubleshooting

### "No weekly plan available"
**Solution**: Click "Generate Weekly Plan" button

### Tasks not showing
**Possible causes**:
- No active roadmap
- All roadmap tasks completed
- Check your goal settings

### Weekly tracker shows gray boxes
**Meaning**: No tasks scheduled for those days
**Action**: Click "Generate Weekly Plan" or complete current tasks

### Task won't complete
**Check**:
- Internet connection
- Try refreshing the page
- Check browser console for errors

## FAQ

**Q: How many tasks per day?**
A: Depends on your daily study time and task duration. Usually 1-3 tasks.

**Q: Can I skip a day?**
A: Yes, but it will break your streak. Tasks remain available.

**Q: What happens to incomplete tasks?**
A: They stay in the system. You can complete them anytime.

**Q: Can I reschedule tasks?**
A: Currently tasks are auto-scheduled. Manual rescheduling coming soon!

**Q: How is the weekly plan generated?**
A: Based on your roadmap, daily study time, and task estimates.

**Q: What if I complete all roadmap tasks?**
A: System will show no tasks until you create a new roadmap/goal.
